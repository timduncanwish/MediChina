# HematologyConnect

> Advanced hematology care coordination service for international patients seeking treatment in North China

## 🌟 Overview

HematologyConnect bridges the gap between international patients and China's leading national hematology center, providing expert coordination services for blood cancers and complex blood disorders.

## ✨ Features

- **Modern Design**: Distinctive UI following frontend-design best practices
- **Responsive**: Fully optimized for desktop, tablet, and mobile devices
- **Interactive Elements**: FAQ accordion, smooth scroll navigation, hover effects
- **Professional Typography**: Space Grotesk + IBM Plex Sans font pairing
- **Unique Color Scheme**: Deep Navy (#0A1628) + Coral Orange (#FF6B4A)

```bash

```

2. Open in browser
```bash
# Simply open index.html in your preferred browser
```

Or use a local server:
```bash

# Node.js (using npx)
npx serve

# Then visit http://localhost:8000
```
